# Kraken Swerve Base
This is a swerve base made for ALL CTRE hardware

## Features
- FOC Support
- CANivore Support
- Simple and barebones

## Future Features
- 250hz odom

## CANcoder Calibration
When calibrating for CANcoder offsets, make sure the bevels face right
